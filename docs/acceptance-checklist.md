# DayRecord MVP 验收清单（PRD §12）

在 Windows 10+ 本机执行。每项记录：通过 / 未通过 / 备注。

| # | 验收项 | 步骤 | 预期 | 结果 |
|---|--------|------|------|------|
| 1 | 首启隐私同意 | 首次启动应用 | 显示全屏同意弹窗；勾选并确认后进入主界面 | ☐ |
| 2 | 英文输入与粘贴 | 打开记事本，输入英文并 Ctrl+V 粘贴 | 今日会话与字符统计增加 | ☐ |
| 3 | 会话落盘 | 切换窗口或停顿 30s | `sessions` 表新增记录 | ☐ |
| 4 | 时间轴 | 切换多个应用工作数分钟 | `activities` 记录各应用停留时长 | ☐ |
| 5 | 空闲不计 | 空闲超过 60s 后继续 | 活跃时长不包含空闲区间 | ☐ |
| 6 | 暂停录制 | 托盘或主界面暂停 | 键盘与粘贴不再写入 | ☐ |
| 7 | 生成复盘 | 配置 API Key，点击「生成今日复盘」 | 输出四段式 Markdown 并保存 | ☐ |
| 8 | 托盘操作 | 左键打开、右键暂停/退出 | 行为符合 PRD | ☐ |
| 9 | 清空数据 | 点击「清空全部数据」 | sessions/activities/summaries/facts 清空；consent 保留 | ☐ |

## 附加（M8 记忆进化层）

| # | 验收项 | 步骤 | 预期 | 结果 |
|---|--------|------|------|------|
| 10 | 巩固记忆 | 有当日数据后点击「巩固记忆」 | `facts` 表新增条目，UI 列表可见 | ☐ |
| 11 | 编辑/删除 fact | 在 UI 点击「改」/「删」 | 数据库与 UI 同步更新 | ☐ |

## 已知限制（文档声明，非失败项）

- IME 中文逐字不完整捕获
- 方向键/鼠标改光标可能导致文本顺序错乱
- 全局键盘钩子可能被杀软误报（建议代码签名）
- 仅支持 Windows
